BOM

BOM是浏览器对象模型
BOM是由一系列相关对象构成，每个对象都提供了很多方法和属性。

BOM里最重要的对象：

  • window (窗口)：window是整个网页的框架，每个网页的内容都是装载在window里面。
  • navigator(浏览器)：navigator里面存储浏览器相关信息
  • history(历史)：history拿来存储整个网页栈的
  • screen(显示屏幕)：screen包含显示屏的信息
  • location(地址)：location包含当前访问的地址(网址)信息
window

window对象表示一个浏览器窗口或一个frame框架它处于对象层次的最顶端，它提供了处理浏览器窗口的方法和属性。
