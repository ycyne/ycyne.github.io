汉诺塔的递归步骤分析

方法设置了4个参数：

  • N：初始N个从小到大的盘子，N是最大编号
  • from：初始的柱子
  • to：目标柱子
  • help：辅助的柱子

代码实现：

步骤分析：以hannuota(3, “A”, “B”, “C”);

步骤 N from to help result
1 3 A B C 运行hannuota(2, from, help,to);等待返回
2 2 A C B 运行hannuota(1, from, help,to);等待返回
3 1 A B C 把1从A移动到B，步骤2递归返回
4 2 A C B 把2从A移动到C
5 2 A C B 运行hannuota(1, help,to,from );等待返回
6 1 B C A 把1从B移动到C，步骤5递归返回
7 2 A C B 步骤1递归返回
8 3 A B C 把3从A移动到B
9 3 A B C 运行hannuota(2, from, help,to);等待返回
10 2 C B A 运行hannuota(1, from, help,to);等待返回
11 1 C A B 把1从C移动到A，步骤10递归返回
12 2 C B A 把2从C移动到B
13 2 C B A 运行hannuota(1, help,to,from );等待返回
14 1 A B C 把1从A移动到B，步骤13递归返回
15 2 C B A 步骤9递归返回
16 3 A B C 结束程序
