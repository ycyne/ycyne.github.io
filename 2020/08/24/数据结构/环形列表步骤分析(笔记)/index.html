环形列表步骤分析(笔记)

自学数据结构与算法：[B站地址]:https://www.bilibili.com/video/BV1E4411H73v

代码细节：

  • 存入数据rear的公式为rear=(rear+1)%maxSize;
  • 取出数据front的公式为front=(front+1)%maxSize;
  • 判断数组状态满(is Full)的条件为(rear+maxSize-front)%maxSize==0
  • 判断数组状态空(is Empty)的条件为rear==front

没有存入数据时

rear=front=0
maxSize=3
这里初始化front和rear为0
front指向队列的第一个元素
rear指向队列的最后一个元素的后一位

存入一个数据arr[0]时

front=0
arr[rear]
使用rear的公式
rear=1
maxSize=3

存入第二个数据arr[1]时

front=0
arr[rear]
使用rear的公式
rear=2
maxSize=3

存入第三个数据arr[2]时

front=0
arr[rear]
使用rear的公式
rear=0
maxSize=3
接着is Full：（rear+maxSize-front）%maxSize==0
这时数组就满了，不能再加了

  • 由于这里的rear相当于轮了一圈到front；（可以类比为大你一转的意思，并不是单纯地0，想象0为360°，）

取出一个数据arr[0]时（先入先出原则）

  1. int value=arr[front];
  2. 使用front的公式
  3. front=1
  4. rear=0,maxSize=3
  5. 还剩下arr[1]、arr[2]
  6. 输出value

取出第二个数据arr[1]时

  1. int value=arr[front];
  2. 使用front的公式
  3. front=2
  4. rear=0,maxSize=3
  5. 还剩下arr[2]
  6. 输出value

取出第三数据arr[2]时

  1. int value=arr[front];
  2. 使用front的公式
  3. front=rear=0，maxSize=3
  4. 输出value
  5. 这时数据已空

再次存入数据

  1. front=0
  2. arr[rear]
  3. 使用rear的公式
  4. rear=1
  5. maxSize=3

环形列表实现分析完成~
